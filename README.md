# Vrinda Store Annual Report 2022 

## 📌 Project Overview
This project is an **Excel dashboard** created to analyze the sales performance of **Vrinda Store in 2022**.  
The goal was to identify customer trends and provide actionable insights to improve sales in 2023.

---
## 🎯 Objectives
- Understand customer demographics and buying behavior
- Identify top-performing states and age groups
- Compare sales across gender and regions
- Provide a clear, interactive dashboard for decision-making

---
## 🔑 Key Insights
- Women buyers contributed **65%** of total sales
- Top states: **Maharashtra, Karnataka, Uttar Pradesh**
- Most active age group: **30–49 years (Adults)**
- Highest sales channel: **Amazon & Flipkart**
